# Manish Jaiswal

17 years building software. Started as an engineer, spent six of those years as a Principal, now running engineering teams.

Most of that time has been in the .NET world — C#, ASP.NET, microservices, event-driven systems. The kind of work where the decisions you make in year one are still running in production in year five, so you learn quickly to think before you type.

Lately I've been deep into AI tooling — specifically how engineers can use Claude Code to actually move faster without cutting corners. Built [claude-commands-kit-pro](https://github.com/manish-7/claude-commands-kit-pro) out of that — 34 slash commands covering the things I found myself prompting repeatedly: code reviews, debugging, security audits, scaffolding, release checks. Also used Claude Code to build some cool AI features in the projects to enhance the productivity and customer experience.
